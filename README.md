**Overview**

This project demonstrates an end-to-end data analytics workflow in the banking and financial services domain. The objective is to analyze customer loan and deposit data to identify portfolio trends, assess lending risk, and generate actionable business insights. The project combines Python for data analysis, MySQL for querying, and Power BI for interactive dashboard development.

Dataset

The dataset contains customer banking information, including demographic details, account balances, deposits, loans, and other financial attributes.

Key fields include:

Customer Demographics (Age, Gender, Nationality)
Estimated Income
Saving Account Balance
Checking Account Balance
Foreign Currency Account
Business Lending
Bank Loans
Bank Deposits
Superannuation Savings

Tools & Technologies
Python (Pandas, NumPy, Matplotlib, Seaborn)
MySQL (Data storage and SQL analysis)
Power BI (Interactive dashboard and business reporting)
Jupyter Notebook
Git & GitHub

**Project Workflow**
1. Data Loading
Imported the dataset into Python.
Examined data structure and variable types.
Identified missing values and duplicate records.
2. Exploratory Data Analysis (EDA)
Analyzed customer demographics and financial behavior.
Examined loan and deposit distributions.
Identified trends, correlations, and outliers.
Generated visualizations to support business insights.
3. Data Cleaning
Removed duplicate records.
Handled missing values.
Corrected inconsistent data types.
Prepared a clean dataset for analysis and reporting.
4. SQL Analysis (MySQL)
Imported the cleaned dataset into MySQL.
Wrote SQL queries to:
Aggregate loan and deposit values
Analyze customer segments
Compare financial products
Generate business KPIs
Support reporting requirements
5. Power BI Dashboard

**Built an interactive dashboard to visualize:**

Total Loan Portfolio
Total Deposits
Customer Distribution
Loan vs Deposit Analysis
Income Segmentation
Gender & Nationality Analysis
Financial Product Distribution
Key Performance Indicators (KPIs)
Dashboard Highlights

**The dashboard provides:**

Interactive slicers for customer segmentation
Dynamic KPI cards
Clustered bar charts
Pie and donut charts
Matrix tables
Trend and comparison visualizations

The dashboard enables users to quickly understand portfolio performance, customer characteristics, and lending patterns.

**Key Results**
Identified major customer segments contributing to loans and deposits.
Analyzed relationships between income levels and banking products.
Highlighted lending trends and potential areas of financial risk.
Demonstrated how data analytics can support informed lending decisions and reduce potential credit risk.
Delivered an interactive dashboard for business reporting and decision-making.
